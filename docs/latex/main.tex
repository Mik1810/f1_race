\begin{abstract}
This document describes the design, architecture, and implementation of a Formula~1 race simulation built on top of the DALI Multi-Agent System framework. Car agents can be dynamically generated from a single agents.json configuration file. A dedicated agent coordinates the entire race flow and injects probabilistic track events (safety car deployment, heavy rain) at runtime. A real-time web dashboard provides live monitoring and manual control of the simulation. Finally, the project is entirely containerized using Docker, allowing for easy setup and execution across different environments.
\end{abstract}
